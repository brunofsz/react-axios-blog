# Blog Com REACT.JS + AXIOS

Este projeto é um blog desenvolvido em React que consome a API de testes JSONPlaceholder usando Axios. Como essa API é apenas simulada, as requisições de POST, PUT e DELETE não alteram realmente os dados no servidor por isso, mesmo após criar, editar ou excluir um post, a listagem original permanece a mesma ao recarregar a página. O objetivo do projeto é demonstrar o funcionamento de requisições HTTP em uma aplicação React, de forma simples e prática.

## Funcionalidades

- Listagem de posts carregados da API JSONPlaceholder
- Criação de novos posts (POST)
- Edição de posts existentes (PUT)
- Exclusão de posts (DELETE)
- Navegação entre páginas usando React Router
- Interface simples e intuitiva construída com componentes reutilizáveis

## Tecnologias Utilizadas

- React
- Axios
- React Router
- JavaScript
- CSS
- Vite

## Como Usar

1. Clone o repositório  
2. Entre na pasta do projeto  
3. Instale as dependências com `npm install`  
4. Execute o projeto com `npm start`
